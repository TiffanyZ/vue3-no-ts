## 功能介绍
项目包括三个页面home、all、list
### home（新增编辑）
主要用于文本编辑和上传（标题、栏目分类、时间、标签、名称、链接、富文本）
其中富文本使用wangeditor，自定义标签save可保存富文本内容（发布后清空）

### all（信息池）
发布后内容进入信息池，新增管理员功能（删除、审批）
点击click可查看内容详情

### list（审批通过信息）
信息池中审批通过进入list，新增管理员功能（撤回）
